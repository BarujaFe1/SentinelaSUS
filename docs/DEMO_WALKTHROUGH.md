# Demo Walkthrough (3–5 minutos)

**Objetivo:** mostrar clareza estatística + responsabilidade, não “dashboard bonito”.

## Roteiro

| Min | Tela | Fale |
|---|---|---|
| 0:00 | Home | “Lab sintético de vigilância responsável — zero dado real.” |
| 0:30 | Overview | KPIs + distribuição de níveis; reliability médio. |
| 1:00 | Explorer | Baseline sazonal + série; “sinal ≠ diagnóstico”. |
| 1:45 | Methodology | z classifica; MAD compara; gráfico dual; limitação do baseline. |
| 2:30 | Comparison | Scatter; concordância com limiares assinados. |
| 3:15 | Simulation | TP/FP/FN vs anomalias plantadas; “falsos alertas são esperados”. |
| 4:00 | Memo | Anti-escopo + linguagem; imprimir se útil. |
| 4:30 | Brief | Achados determinísticos sem termos proibidos. |

## Perguntas que você quer receber

- Por que z e não só MAD?
- O que é um falso positivo aqui?
- Por que a baseline inclui o ano avaliado?
- Como isso mudaria com dados reais (LGPD, auth, leave-one-out)?
